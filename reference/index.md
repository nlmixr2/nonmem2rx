# Package index

## Importing NONMEM to rxode2

- [`nonmem2rx()`](nonmem2rx.md) : Convert a NONMEM source file to a
  rxode model (nlmixr2-syle)

## Cross-checking manual models

- [`as.nonmem2rx()`](as.nonmem2rx.md) : Convert a model to a nonmem2rx
  model

## Reading in information from `NONMEM`

- [`nmcov()`](nmcov.md) : Read in data file

- [`nmext()`](nmext.md) :

  Reads the NONMEM `.ext` file for final parameter information

- [`nmgrd()`](nmgrd.md) :

  Reads the NONMEM `.grd` file for final parameter gradient

- [`nminfo()`](nminfo.md) : Get the most accurate information you can
  get from NONMEM

- [`nmlst()`](nmlst.md) :

  Reads the NONMEM `.lst` file for final parameter information

- [`nmtab()`](nmtab.md) : Read nonmem table file

- [`nmxml()`](nmxml.md) :

  Read a nonmem xml and create output similar to the
  [`nmlst()`](../reference/nmlst.md)
