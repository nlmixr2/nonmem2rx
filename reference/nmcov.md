# Read in data file

Read in data file

## Usage

``` r
nmcov(file, ...)
```

## Arguments

- file:

  file name to read the results from

- ...:

  other parameters passed to
  [`data.table::fread`](https://rdrr.io/pkg/data.table/man/fread.html)

## Value

A matrix with covariance step from NONMEM

## Author

Philip Delff and Matthew L. Fidler
